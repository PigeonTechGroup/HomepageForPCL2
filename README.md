---

## 食用方法

1. 打开 PCL2 ，点击 设置 -> 个性化 -> 在 主页 栏选择 “ 读取本地文件 ”。
2. 打开与 PCL2启动器 同级的 PCL 文件夹，丢入 Custom.xaml 。
3. 刷新 PCL2 主页，或重新打开 PCL2 页面。
4. ~~你吃完了~~
